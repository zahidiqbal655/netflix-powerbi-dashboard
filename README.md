# Netflix Content Analysis — SQL + Power BI 🎬

## Overview
End-to-end data analysis project on 9,000+ Netflix titles using SQL for 
data exploration and Power BI for interactive dashboard visualization.

## Dashboard Preview


![Netflix Dashboard](Screenshot%202026-07-12%20033124.png)



## Project Structure
- 📄 SQL Queries — 15+ queries for data analysis
- 📊 Power BI Dashboard — Interactive visualization
- 📁 Dataset — Netflix Movies and TV Shows (Kaggle)

## Key Insights
- United States leads with 3.2K titles
- TV-MA is the most common rating
- Content releases peaked around 2019-2020
- Movies dominate at 69.69% of total content

## Tools Used
- MySQL
- Power BI Desktop
- Power Query
- DAX Measures

## Dataset
- Source: Kaggle — Netflix Movies and TV Shows
- Records: 9,000+ titles
